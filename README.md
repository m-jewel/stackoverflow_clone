# About
  StackOverflow Clone is a project that aims to replicate the core functionalities of StackOverflow, where it is a platform for users to ask questions, share answers, and engage with the developer or in general community.

# Features
  **1. User Authentication:** Users can sign up and log in to access all the features of the site.<br>
  **2. Post Questions:** Logged-in users can post their own questions to get help from the community.<br>
  **3. Comment and Answer:** Users can comment on questions and provide answers to help others.<br>
  **4. Upvote and Downvote:** Questions and comments can be rated with upvotes and downvotes to highlight the best q/a'.<br>
  **5. User Profiles:** Users can access their profile and change names and passwords.
  **6. Search and Filter Functionality:**  Efficiently search for questions and answers.

# Tech Stack
  **Frontend:** HTML, CSS, JavaScript, React<br>
  **Backend:**  Node.js, Express<br>
  **Database:** MySQL<br>
  **Authentication:** JWT (JSON Web Tokens)<br>
  **Containerization:** Docker<br>
    
# Prerequisites
  Docker

# How to Run
  ### 1. Clone the repository:
    git clone https://github.com/m-jewel/stackoverflow_clone.git
    cd stackoverflow_clone

  ### 2. Start the Docker containers:
    docker-compose up --build

  ### 3. Access the application:
    - Frontend: Open your browser and go to http://localhost:3000
    - Backend: To start the db/API server, open another browser and go to http://localhost:8080

# Usage
After starting the Docker containers, you can interact with the StackOverflow Clone by signing up, asking questions, answering them, and exploring other functionalities.
